# CodeAlpha_LanguageTranslation

A lightweight desktop translation tool built with Tkinter and deep-translator. It auto-detects the source language and translates into a selected target language.

## Features
- Auto source-language detection
- Quick language selection (Spanish, French, German, Japanese)
- Simple, clean GUI

## Installation
1. Create and activate a virtual environment.
2. Install dependencies:
   - `deep-translator`

## Usage
Run the app:
- `python translator_app.py`

## Technologies Used
- Python
- Tkinter
- deep-translator
